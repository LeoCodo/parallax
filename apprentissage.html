Ce que j'ai appris dans le cours R3.03

I) HTML

Dans ce cours, j'ai appris les bases du langage HTML pour structurer des pages web. Voici les principales notions que j'ai étudiées :

  • Structure de base d'une page HTML : La syntaxe de base incluant les balises <html>, <head>, <body>.
  • Les balises HTML essentielles : <header>, <footer>, <section>, <article>, etc.
  • Création de liens : Utilisation de la balise <a> pour créer des liens hypertextes.
  • Les listes : Création de listes ordonnées (<ol>) et non ordonnées (<ul>), ainsi que les éléments de liste (<li>).
  • Les formulaires : Utilisation des balises <form>, <input>, <textarea>, <button> pour créer des formulaires interactifs.

II) CSS

Le CSS permet de styliser les pages HTML et d'améliorer leur présentation. Voici ce que j'ai appris sur le CSS :

  • Les sélecteurs CSS : Sélectionner des éléments HTML avec des sélecteurs comme les classes, les identifiants et les balises.
  • Les propriétés de mise en page : Propriétés telles que display, margin, padding, border, position, etc.
  • Le modèle de boîte (box model) : Comprendre comment les marges, bordures, paddings et dimensions d'un élément interagissent.
  • Les couleurs et arrière-plans : Application des couleurs avec color, background-color, et gestion des arrière-plans avec background-image.
  • Les polices et typographie : Utilisation de propriétés telles que font-family, font-size, line-height pour définir les polices de texte.
  • Les transitions et animations : Création d'animations simples avec transition et @keyframes pour rendre les pages plus dynamiques.
